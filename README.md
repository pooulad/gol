### Run in go:

Make sure you installed all dependencies in https://ebitengine.org/

After install all required tools run:

```bash
    git clone https://github.com/pooulad/gol.git
```
```bash
    cd ./gol/go
```
```bash
     make run
```
OR
```bash
     go run life.go
```
